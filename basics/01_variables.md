
### Example 1: Integer Variable

#### Elixir Code

```elixir
# example1.exs
age = 30                                  # Define an integer variable
IO.puts "Age: #{age}"                     # Print the value of the variable
```

#### Output

```shell
elixir example1.exs
Age: 30
```
